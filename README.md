**Vrinda Store  Sales Dashboard(Excel)**

*Project Overview*
This Project analyzes sales performance of Vrinda store using MS Excel.

**Tools Used**
Excel : 1.Pivot Tables
        2.Pivot Charts
        3.Slicers

**Dashboard Feautures**
1.Sales vs Orders Trend
2.Sales by Gender
3.Orders by Channel
4.Order Status Analysis
5.Top 5 States Analysis
6.Age vs Gender Analysis

**Key Insights**
1.Women contribute the majority of sales.
2.Amazon is the leading sales channel.
3.Maharastra generates the highest sales.
4.Adults represent the largest customer segment.
